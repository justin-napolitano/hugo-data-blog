This repo is a Hugo-based static blog focused on data-driven content. It tackles topics like energy sectors, AI in legal research, and carbon capture economics, all styled with the Anubis theme.

### Key Features:
- **Static site** generated with Hugo
- **Comments** via Disqus, Isso, or Utterances
- **Google Analytics** included
- **Social media** integrations
- Supports rich content with Python analysis and geospatial visualizations

### Tech Stack:
- Hugo, HTML, YAML
- Python, GeoPandas, Folium, Matplotlib for data work
- Neo4j for graph database discussions

### Getting Started:
1. Clone and navigate to the repo:
   ```bash
   git clone https://github.com/justin-napolitano/hugo-data-blog.git
   cd hugo-data-blog
   ```
2. Start the Hugo server:
   ```bash
   hugo server -D
   ```
3. Open your browser at `http://localhost:1313`.

**Gotcha:** Check `config.yaml` for your site's metadata before deploying.
